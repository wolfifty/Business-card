Обо мне

Константин Малышев

Разрабатываю IT-продукты для экспертов, коучей и блогеров — от Telegram-ботов с воронками продаж до полноценных Mini Apps с оплатой и личным кабинетом.

Работаю на стыке no-code и классической разработки: беру Salebot там, где нужна скорость и гибкость без лишних затрат, и перехожу на Python и React, когда задача требует кастомной логики, интеграций или собственного интерфейса.

За плечами — десятки ботов и мини-приложений: квизы с весовой логикой подбора, автоматизация приёма заявок, системы с оплатой через YuKassa, интеграции с Claude API для генерации контента. Понимаю специфику инфобизнес-ниши изнутри — как выстроена воронка эксперта, что действительно упрощает жизнь клиенту, а что становится лишней технической нагрузкой.

Работаю от идеи до запуска: обсуждаем задачу, собираю прототип, дорабатываем вместе, сдаю рабочий продукт с поддержкой после запуска.

Стек Salebot, Python (aiogram), React, PostgreSQL, Claude API
